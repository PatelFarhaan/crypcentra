# _**CRYPCENTRA**_

## How to run this application:

### Run the application:
Build the image using following command:
```
docker build -t crypcentra:latest .
```

Run the docker image using the command:
```
docker run -it --name crypcentra -p 8000:8000 crypcentra
```
